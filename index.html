Jawaban 1

Karena sebelumnya sudah diketahui bahwa data memiliki kecenderungan berbentuk y = ab^x maka metode linearisasi yang digunakan adalah Linearisasi Pangkat Sederhana.

Persamaan y = ab^x selanjutnya akan diubahsuaikan dengan persamaan y = Cx^b

Langkah melakukan Linearisasi Pangkat Sederhana

  1. Menyesuaikan data dengan fungsi y = Cx^b
  2. Mendefinisikan nilai dari masing-masing parameter: Y = a + bX
    Y = ln(y),
    a = ln(C),
    X = ln(x)
  3. Mendefinisikan nilai x[i] dan y[i] (sudah diketahui dari tabel)
  4. Mencari nilai:
    X[i] = ln(x[i]),
    Y[i] = ln(y[i]),
    (X[i])^2, dan (X[i] * Y[i])
    Serta memperoleh hasil perhitungan:
    ∑(X[i]), ∑(Y[i]), ∑((X[i])^2), dan ∑(X[i] * Y[i])
  5. Lakukan penyesuaian data yang sebelumnya bukan merupakan data linear dengan menggunakan sistem persamaan linear:
    (n * a) + (∑(X[i]) * b) = ∑(Y[i])
    (∑(X[i]) * a) + ((∑(X[i]))^2 * b) = ∑(X[i] * Y[i])

    n = jumlah data,
    a dan b = parameter yang dicari
  6. Setelah dilakukan perhitungan diperoleh nilai parameter a dan b.
  7. Cari nilai C melalui persamaan C = e^a
    e = 2.718281828459045 (konstanta euler)
  8. Model akan diperoleh melalui persamaan: y = C*(x^b)
